## Notepad

Program for writing memos, bookmarks and tasks.

Written in Ruby.

### Adding posts

To add a new post, enter in the terminal:

```
ruby add_post.rb
```

Posts are saved to `notepad.db` using the` sqlite3` gem

### Viewing Notepad

To view the posts, type in the terminal:

```
ruby notepad.rb [options]
```

For help:

```
ruby notepad.rb -h
```
